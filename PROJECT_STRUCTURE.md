## рҹ“Ҡ Component Responsibilities

### рҹҺҜ Core System (`core/`)
**Primary Role**: Application foundation and main logic
- **main.py**: Entry point, argument parsing, system initialization
- **gesture_app.py**: Main application loop, component coordination
- **hand_tracker.py**: Gesture detection with neural integration

### рҹ§  Neural Engine (`neural_engine/`)
**Primary Role**: AI-powered gesture enhancement
- **nami_neural_engine.py**: Core neural processing algorithms
- **neural_config_manager.py**: Performance profile management
- **neural_system_monitor.py**: Real-time system monitoring

### рҹӨ– AI Intelligence (`ai_intelligence/`)
**Primary Role**: Learning and intelligent suggestions
- **nami_ai_intelligence.py**: Pattern learning and word prediction
- **ai_virtual_keyboard.py**: Smart keyboard with AI integration

### рҹҺЁ UI Components (`ui_components/`)
**Primary Role**: User interface and visual feedback
- **virtual_keyboard.py**: Standard keyboard implementation
- **welcome_screen.py**: Application introduction screen

### рҹӣ пёҸ Utilities (`utils/`)
**Primary Role**: Support functions and error handling
- **error_handler.py**: System reliability and error recovery
- **logger.py**: Session tracking and data persistence
- **text_to_speech.py**: Audio feedback system

### рҹ’ҫ Cache (`cache/`)
**Primary Role**: Runtime data and system-generated files
- **nami_ai_data.json**: AI learning data in human-readable format
- **nami_ai_data.pkl**: AI learning data in binary format for fast loading
- **gesture_keyboard_log.txt**: Application runtime logs and session data

### вҡҷпёҸ Configuration (`config/`)
**Primary Role**: System settings and dependencies
- **requirements.txt**: Python package dependencies
- **gesture_keyboard_log.txt**: Legacy log file (moved to cache/)

### рҹҺ® Demos (`demos/`)
**Primary Role**: System demonstration and testing
- **neural_demo.py**: Comprehensive feature demonstration

### рҹ“ҡ Documentation (`docs/`)
**Primary Role**: User and developer guidance
- **README.md**: Basic usage documentation
- **NAMI_NEURAL_README.md**: Advanced feature documentation

- **Memory Management**: Automatic optimization and cleanup
- **Performance Profiles**: Adaptive to system capabilities

   - Profile Settings, Parameters
   - Performance Thresholds
   - User Customizations
   - Hardware Capabilities
